## Managing Expenses

### View Expenses
1. Click **Expenses** in the sidebar
2. Browse your expense list
3. Use the **search bar** to find specific expenses
4. Use the **category filter** to view expenses by category

### Add an Expense
1. Click **Add Expense** button
2. Fill in the form:
   - **Name**: Select from dropdown or type custom name
   - **Category**: Choose a category (Food & Dining, Transportation, etc.)
   - **Amount**: Enter the amount (must be positive)
   - **Date**: Select the date (defaults to today)
   - **Business Tags** (optional): Select tags to categorize this expense
3. Click **Add Expense**

### Edit an Expense
1. Click the **Edit** (✏️) icon next to an expense
2. Modify any field
3. Update business tags if needed
4. Click **Update**

### Delete an Expense
1. Click the **Delete** (🗑️) icon next to an expense
2. Confirm deletion

## Business Section - Tag Your Expenses

The Business section helps you organize and analyze expenses using custom tags.

### What Are Business Tags?
Tags are custom labels you create to categorize expenses (e.g., "Essential", "Marketing", "Travel"). You can assign multiple tags to any expense to track spending across different categories.

### Creating Tags

1. Click **Business** in the sidebar
2. Click **Manage Tags** button
3. Click **Create New Tag**
4. Enter tag details:
   - **Name**: e.g., "Essential", "Marketing", "Health"
   - **Color**: Choose a color for visual identification
5. Click **Create Tag**

**Example Tags:**
- 🏷️ Essential (#FF5722 - Orange)
- 🏷️ Health (#4CAF50 - Green)  
- 🏷️ Marketing (#2196F3 - Blue)
- 🏷️ Travel (#9C27B0 - Purple)

### Tagging Expenses

#### Tag a New Expense
1. When creating an expense, scroll to **Business Tags (optional)**
2. Check the boxes next to tags you want to apply
3. You can select multiple tags
4. Complete the expense form and save

#### Tag an Existing Expense
1. Edit the expense (click ✏️ icon)
2. In the **Business Tags** section:
   - Check tags you want to add
   - Uncheck tags you want to remove
3. Click **Update**

**Important:** You can update tags as many times as needed without errors!

### Visual Indicators for Tagged Expenses

Tagged expenses are easy to spot:
- **💼 BIZ Badge** appears next to the expense name
- **Business Corner Ribbon** shows 💼 icon in top-right corner
- **Tag Count** displays how many tags are applied

### Filtering by Tags

1. In the Expenses list, look for the **Filter by Tag** dropdown
2. Select a tag to see only expenses with that tag
3. Select "All Expenses" to clear the filter

### Business Analytics Dashboard

View detailed insights about your tagged expenses:

1. Click **Business** in the sidebar
2. The analytics dashboard shows:

**Summary Cards:**
- Total Expenses (sum of all tagged expenses)
- Active Tags (number of tags in use)
- Total Transactions (count of tagged expenses)

**Spending by Tag:**
- Bar chart showing how much you spent per tag
- Helps identify major expense categories

**Category Breakdown:**
- Your expenses grouped by category
- Shows which tags are used in each category

**Recent Tagged Expenses:**
- List of your latest expenses with their tags
- Quick overview of recent business activity

**Monthly Tag Report:**
- Timeline showing tag usage over months
- Spot trends in your spending patterns

### Managing Tags

#### Edit a Tag
1. Go to **Business** → **Manage Tags**
2. Click the **Edit** (✏️) icon next to a tag
3. Change the name or color
4. Click **Update**

**Note:** Editing a tag updates it everywhere it's used!

#### Delete a Tag
1. Go to **Business** → **Manage Tags**
2. Click the **Delete** (🗑️) icon next to a tag
3. Confirm deletion

**Note:** Deleting a tag removes it from all expenses, but the expenses themselves remain intact.

### Tag Best Practices

✅ **DO:**
- Create meaningful tag names ("Essential", "Tax Deductible", "Recurring")
- Use colors to visually group related tags
- Tag expenses consistently for accurate analytics
- Use multiple tags when an expense fits multiple categories

❌ **DON'T:**
- Create too many similar tags (e.g., "Marketing1", "Marketing2")
- Use generic names that don't help you analyze spending
- Forget to tag important business expenses

### Common Workflows

#### Scenario 1: Track Essential Business Expenses
1. Create "Essential" tag
2. Tag all must-have expenses (rent, software, insurance)
3. View analytics to see total essential spending
4. Use this data for budgeting

#### Scenario 2: Prepare for Tax Season
1. Create "Tax Deductible" tag
2. Tag all deductible expenses throughout the year
3. Filter by "Tax Deductible" tag
4. Export list for tax preparation

#### Scenario 3: Department Budget Tracking
1. Create tags for each department (Marketing, IT, Operations)
2. Tag expenses accordingly
3. View Spending by Tag chart to compare departments
4. Identify areas for cost optimization

### Troubleshooting

**Q: I updated an expense's tags but got an error.**
**A:** This has been fixed! You can now update expense tags as many times as you want without errors.

**Q: Can I use someone else's tags?**
**A:** No, tags are private to each user. You can only see and use your own tags.

**Q: What happens to expenses if I delete a tag?**
**A:** The tag is removed from all expenses, but the expenses themselves remain. Only the tag association is deleted.

**Q: Can I have expenses without tags?**
**A:** Yes! Tags are optional. You can create expenses without any tags and add them later.

**Q: How many tags can I add to one expense?**
**A:** There's no hard limit, but we recommend 2-3 tags per expense for clarity.
